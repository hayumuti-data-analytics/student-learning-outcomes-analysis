Student Learning Outcomes Analysis

Role: Educational Data Analyst

Context: Portfolio project demonstrating Excel-based analysis to support academic and institutional decision-making.

Project Overview

This project demonstrates an end-to-end Excel-based educational data analysis workflow, from raw assessment data to actionable academic insights.

Research Questions

  • How do students perform across different assessment components?
  • What is the overall distribution of student achievement?
  • Which components contribute most to final learning outcomes?

Deliverables

Key Skills Demonstrated

  • Analysis Educational data analysis for learning outcomes evaluation
  • Excel Advanced Excel formulas and logical modeling
  • Aggregation Pivot tables for performance summarization
  • Visualization Dashboard design for academic decision support

Key Insights

Executive Summary:

  • Final exam scores show the strongest contribution to overall student achievement compared to quizzes and assignments.
  • Score distribution indicates moderate variance, suggesting relatively consistent learning outcomes across students.
  • Strong assignment performance correlates with stable final grades, highlighting the role of continuous assessment.
