Student Learning Outcomes Analysis

Role: Educational Data Analyst
Context: Excel-based portfolio project for academic decision-making

Project Overview

This project demonstrates an end-to-end Excel-based educational data analysis workflow, transforming raw student assessment data into actionable academic insights for institutional decision-making.

Research Questions

  • How do students perform across different assessment components?
  • What is the overall distribution of student achievement?
  • Which components contribute most to final learning outcomes?

Deliverables

  • Raw Assessment Dataset (PDF): Download
  • Final Excel Analysis File: Download

Key Skills Demonstrated

  • Analysis Educational data analysis for learning outcomes evaluation
  • Excel Advanced Excel formulas and logical modeling
  • Aggregation Pivot tables for performance summarization
  • Visualization Dashboard design for academic decision support

Key Insights

  • Final exam scores show the strongest contribution to overall student achievement.
  • Score distribution indicates moderate variance, suggesting relatively consistent learning outcomes.
  • Strong assignment performance correlates with stable final grades, highlighting continuous assessment.
